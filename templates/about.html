About

Welcome to the About page of our Object Detection application.

Overview

Our Object Detection application allows users to upload images and detect objects within those images using computer vision techniques.

Features

  • Upload images for object detection
  • Detect objects within uploaded images
  • Display processed images with detected objects

Technologies Used

Our application is built using the following technologies:

  • Python Flask for the backend
  • OpenCV for image processing
  • Bootstrap for frontend styling

How to Use

To use our Object Detection application, follow these steps:

  1. Visit the home page to upload an image for object detection.
  2. Upload an image using the provided form.
  3. Once the image is uploaded, the application will process it and display the detected objects.
  4. You can view the processed image with detected objects on the same page.
